# Car4Race

> 汽车爱好者的数字化服务平台

## 项目简介

Car4Race 是一个面向汽车爱好者的综合服务平台，包含多个独立子应用，共用技术栈和基础设施。

## 子应用列表

| 子应用 | 说明 | 状态 |
|--------|------|------|
| [私域视频网站](./docs/私域视频网站/) | 个人知识分享者的私域网站，支持笔记展示、课程销售、会员体系 | 🟡 开发中 |
| [圈速榜](./docs/圈速榜/) | 面向赛道日爱好者的圈速记录与社区榜单平台 | ⚪ 待开始 |

## 技术栈

所有子应用共用以下技术栈：

| 层次 | 技术 |
|------|------|
| 后端 | Go + Gin + GORM |
| 数据库 | SQLite |
| 前端 | Vue 3 + Vite + TailwindCSS |
| 部署 | Docker + Nginx |
