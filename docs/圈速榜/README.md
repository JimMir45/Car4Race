# Car4Race 圈速榜 - 文档导航

## 项目简介

**Car4Race 圈速榜** 是 Car4Race 项目的子应用，面向赛道日爱好者的个人圈速记录本 + 车友圈速社区。

### 核心功能

- **个人记录**：记录每次练习成绩，管理个人最佳
- **GPS计时**：山路起终点自动计时
- **数据导入**：支持 RaceChrono 数据导入
- **综合榜单**：聚合官方/大V/个人成绩
- **投票验证**：社区投票验证成绩真伪
- **山路标记**：用户自定义山路起终点

### 与 HPA 的关系

| 方面 | 说明 |
|------|------|
| 用户体系 | 共用 |
| 技术栈 | 共用（Go + Vue3 + SQLite） |
| 部署 | 可独立部署或整合 |

## 快速了解

### 成绩记录流程

```
用户记录成绩 → 是否公开上榜？
                    │
        ┌───────────┴───────────┐
        │                       │
       否                      是
        │                       │
        ▼                       ▼
   保存私密记录          检查照片+视频
                              │
                              ▼
                        投票验证（10人）
                              │
                    ┌─────────┴─────────┐
                    │                   │
                  <5假                ≥5假
                    │                   │
                    ▼                   ▼
                成绩上榜           成绩作废+禁榜
```

### 榜单数据组成

```
综合榜单
├── 官方成绩（爬取）
│   ├── 51GT3
│   └── 赛道官方微博
├── 大V成绩（爬取）
│   ├── 键盘车神教
│   └── 懂车帝实测榜
└── 个人成绩（用户上传）
    └── 需投票验证通过
```
