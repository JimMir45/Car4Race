# 个人私域学习网站 - HPA (Home Private Academy)

> 一个轻量级的个人学习笔记分享与付费课程下载平台

## 项目定位

面向个人知识分享者的私域网站，支持：
- **免费内容**：学习笔记公开展示，游客可浏览
- **付费内容**：视频课程打包下载，注册用户付费后下载
- **会员权益**：年消费满 2000 元解锁全站下载权限（有效期1年）

## MVP 核心功能

| 功能模块 | 说明 | 优先级 |
|---------|------|--------|
| 笔记展示 | Markdown 笔记渲染，游客可浏览 | P0 |
| 用户注册 | 手机号+验证码快速注册，用户名自动生成（如202601156826） | P0 |
| 课程列表 | 付费课程目录，支持多种排序（点击量/下载量/更新时间） | P0 |
| 支付系统 | 第三方支付（预留）+ 邀请码兑换 | P0 |
| 个人中心 | 查看订单、浏览记录、下载记录 | P0 |
| 会员体系 | 年消费满 2000 元获全站下载权限，公告滚动播放3日 | P1 |
| 安全防护 | 用户级访问频率限制（15秒）、游客IP排队（>50排队） | P0 |

## 技术选型

- **前端**：Vue3 + Vite（轻量快速）
- **后端**：Go + Gin（单文件部署）
- **数据库**：SQLite（零运维）
- **部署**：Docker 容器化
- **短信服务**：阿里云/腾讯云短信（验证码）

## 快速开始

```bash
# 拉取镜像
docker pull your-registry/hpa:latest

# 启动服务
docker run -d \
  -p 8080:8080 \
  -v /path/to/notes:/app/static/notes \
  -v /path/to/courses:/app/static/courses \
  -v /path/to/data:/app/data \
  --name hpa \
  your-registry/hpa:latest
```

## 核心业务规则

| 规则 | 说明 |
|-----|------|
| 用户名生成 | 日期(YYYYMMDD) + 手机号后4位，如 `202601156826` |
| 访问频率 | 注册用户每 15 秒限制一次接口调用 |
| 游客排队 | 在线游客 IP 超过 50 时，新游客需排队 |
| 会员触发 | 年消费（不含邀请码）满 2000 元 → 全站下载权限 |
| 会员有效期 | 从达标时刻起，1 自然年 |
| 会员公告 | 新会员滚动公告 3 天 |
